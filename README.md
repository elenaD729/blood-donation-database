Academic database project designed to model and manage a blood donation system.

This project focuses on designing and implementing a relational database for managing blood donation activities. The database stores and organizes information about donors, their medical record, blood types, donations, hospitals, patients and transfusions, ensuring data consistency and logical relationships between entities.

The project was developed without a graphical user interface and emphasizes database structure and querying.

Relational database design
Use of primary keys and foreign keys
